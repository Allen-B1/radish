New Variant: Part 1

What to Upload

  1. A map in .svg format with the following features:
    1. The id of each land tile is set to the province abbreviation
    2. The id of each supply center marker is set to sc-Abbr where Abbr is the province abbreviation
    3. For each province there is an object with id unit-Abbr where Abbr is the province abbreviation
    4. For each coast there is an object with id unit-Abbr-xc where Abbr is the province abbreviation and xc is the coast name
  2. A province adjacency .json file.
  3. (Optional) A metadata .json file. If not included, the information will be filled out on the next page.
